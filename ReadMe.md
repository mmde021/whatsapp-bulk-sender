# WhatsApp Bulk Sender

A desktop application built with Python and PyQt5 for sending WhatsApp messages to multiple phone numbers using WhatsApp Web.

## Features

- Send messages to multiple numbers
- Automatic Iranian phone number normalization
- QR login support
- Adjustable delay between messages
- Colored logs
- Stop sending at any time
- Windows executable (.exe)

## Technologies

- Python
- PyQt5
- Selenium
- WebDriver Manager
